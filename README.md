# DataBricks_2026
This repo will be used to showcase my databricks learning and few hands-on projects.
# Architecture of the repository 
-Raw data gathered from various sources staged into Bronze layer.\
-In Silver layer Data is cleaned and stored into SQL tables\
-In Gold layer data is modeled to get the facts and dimensions to apply business logic.\
-To run all of these notebooks on a particular schedule Job ochestration is designed.\
-merged into Git to apply CICD process.
